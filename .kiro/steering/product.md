# プロダクト概要

**otak-lcp**は、VSCode向けの日本語文法解析拡張機能です。MeCabなどの外部依存なしで、kuromoji.jsを内蔵し、日本語テキストの文法チェック、セマンティックハイライト、ホバー情報表示を提供します。

## コア機能

1. **日本語文法チェック**: 二重助詞、助詞連続、ら抜き言葉、文体混在など50種類以上の文法ルールを検出
2. **セマンティックハイライト**: 品詞（名詞、動詞、形容詞、助詞、副詞）に基づく色分け表示
3. **ホバー情報**: 単語の品詞情報、原形、読みを表示
4. **多言語対応**: Markdown、JavaScript、TypeScript、Python、C/C++、Java、Rustのコメント内日本語を解析

## ターゲットユースケース

- **技術文書の校正**: 技術用語の表記統一（JavaScript、AWS、ChatGPT等）
- **日本語ライティング支援**: 弱い表現、冗長表現、読点過多の検出
- **プログラマー向け**: コードコメント内の日本語チェック

## 価値提案

- **外部依存なし**: MeCabやMeCab辞書のインストール不要
- **npm installのみ**: kuromoji.js（IPA辞書内蔵）で即座に利用可能
- **LSP準拠**: Language Server Protocolによる高い互換性と拡張性

## 文法ルールカテゴリ

### 基本ルール
- 二重助詞検出（「がが」「をを」）
- 助詞連続検出（「がを」「をが」）
- 動詞-助詞不整合検出

### 高度なルール
- 文体混在検出（敬体/常体）
- ら抜き言葉検出
- 技術用語表記統一
- 漢字開き検出
- 冗長表現・重複表現検出

### Markdown構造ルール
- 見出しレベル飛び検出
- テーブル列数不一致検出
- コードブロック言語指定欠落検出
